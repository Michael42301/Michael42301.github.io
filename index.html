My Website for Web Class Assignments

Website is designed to show fun facts about dogs. My name is Michael and I'm currently taking a web design class which focuses on this website for our week 4 assignment. I chose to do the topic on dogs because I enjoy dogs and animals in general, I also have a dog myself. I hope this little description gives some more information about myself.

Fun Facts about Dogs

  • Dogs are as smart as a two-year old child.
  • Dogs can get jealous if you give attention to other dogs.

  • Petting dogs can lower blood pressure.
